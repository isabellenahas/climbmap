# Climb Map V2 — pacote de código compatível

Este ZIP foi produzido a partir do repositório real enviado, não de uma estrutura genérica.

## Publicação segura

1. Faça backup da branch atual.
2. Crie `feature/v2-base`.
3. Substitua os arquivos pelo conteúdo deste pacote.
4. Execute `node scripts/validate-data.mjs`.
5. Teste localmente com servidor HTTP.
6. Publique primeiro o código e os CSVs convertidos deste pacote.
7. Depois substitua os CSVs pelos documentos definitivos, preservando os contratos do manifesto.

A pasta `data/legacy-v1` contém cópia dos arquivos recebidos para comparação e recuperação.
