# Climb Map — Starter

Primeiro esqueleto navegável do Climb Map, preparado para GitHub Pages.

## O que já funciona

- criação e entrada em perfil local com nome único e PIN;
- primeiro perfil criado como administrador;
- navegação entre as telas principais;
- modo claro e escuro;
- armazenamento local centralizado;
- exportação de backup técnico em JSON;
- importação substitutiva de backup;
- estrutura modular para receber os dados publicados do Google Sheets.

## Como abrir

Como o projeto usa módulos JavaScript, execute em um servidor local ou publique pelo GitHub Pages. No GitHub, basta subir todos os arquivos para a raiz do repositório e ativar **Settings → Pages → Deploy from a branch**.
